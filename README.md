# Kurulum:

## Android(Termux):

```console

apt update && apt upgrade

pkg install python git

git clone https://github.com/Lowzy1337/Discord-Nuke-Bot

cd Discord-Nuke-Bot

python -m pip install -r requirements.txt

python nuke-bot.py

```

**[Sonraki lansmanlar]**

```console

cd Discord-Nuke-Bot

python nuke-bot.py

```

## Linux / Windows

```console

git clone https://github.com/Lowzy1337/Discord-Nuke-Bot

cd Discord-Nuke-Bot

python -m pip install -r requirements.txt

python3 nuke-bot.py

```
